Custom Select

It's a free light weight very simple jQuery plugin that allows you to create a custom select with images. Supports IE7+

This plugin use minimum css and very simple in customization.
Also you can add images.

DOWNLOAD

Demos:

1. Just select

Value: 

    <select name="my-select" id="my-select">
        <option value="usa">USA</option>
        <option value="russia">Russia</option>
        <option value="england">England</option>
        <option value="italy">Italy</option>
    </select>

    $('#my-select').customSelect();

2. Select with images

Value: 

    <select name="my-select" id="my-select">
        <option data-icon="images/usa.png" value="usa">USA</option>
        <option data-icon="images/ru.png" value="russia">Russia</option>
        <option data-icon="images/en.png" value="england">England</option>
        <option data-icon="images/italy.png" value="italy">Italy</option>
    </select>

    $('#my-select').customSelect( {defaultValue: 'Choose country'} );

3. Select with images and fixed drop down height

Value: 

    <select name="my-select" id="my-select">
        <option data-icon="images/usa.png" value="usa">USA</option>
        <option data-icon="images/ru.png" value="russia">Russia</option>
        <option data-icon="images/en.png" value="england">England</option>
        <option data-icon="images/italy.png" value="italy">Italy</option>
    </select>

    $('#my-select').customSelect({height: '129px'});

Parameters:

defaultValue - Set default text to display when no option is selected.

height - Height in px for the drop down options.

Methods:

update - Update custom select. Use when optons your basic select is changed. You can set height, if its need.(like this $('#my-select').customSelect({'update', '129px'}); )
